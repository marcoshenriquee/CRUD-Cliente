# 🧑‍💼 CRUD de Clientes

Gerenciador simples de clientes desenvolvido em Python, utilizando armazenamento em JSON.  
Permite cadastrar, listar, buscar, editar, excluir e limpar registros de clientes.

## 🚀 Funcionalidades
- ➕ Cadastrar clientes
- 📑 Listar todos os clientes
- 🔎 Buscar clientes pelo nome
- 🔄 Editar informações dos clientes
- ➖ Excluir clientes
- 🆑 Limpar todos os clientes cadastrados

## 🛠 Tecnologias utilizadas
- Python 3.x
- JSON (armazenamento dos dados)
- Bibliotecas padrões do Python (`os`, `json`, `re`)
